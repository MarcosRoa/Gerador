⚠️ Regras de Validação

  • Separador: ponto e vírgula (;)
  • Primeira coluna: data no formato DD/MM/AAAA
  • Números: 2 dígitos (01, 02, ..., 60)
  • Não pode haver linhas vazias no meio do arquivo
  • Timemania: a última coluna após os números é o nome do time

🔄 Pipeline de Processamento

1. Upload via interface ou fetch(`/csv/${lottery}.csv`)
2. Parsing do CSV (detecta separador automaticamente)
3. Extração das datas para filtro por período
4. Validação dos números (range e unicidade)
5. Cache em memória para evitar recarregamentos
6. Alimentação da IA para treinamento
