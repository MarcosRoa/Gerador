🔄 Fluxo de Dados (v7.0)

Web (index.html)
    ├── api-client.js → API (Vercel Functions)
    │       ├── /api/user/credits → credits.service.ts → user.repository.ts
    │       ├── /api/generate → generate.service.ts → game.repository.ts
    │       └── /api/pro/status → auth.service.ts → user.repository.ts
    ├── auth.js → Firebase Auth
    ├── jogos.js → IA local + API
    └── user-interface.js → API para créditos

API (TypeScript) → Services → Repositories → Supabase (RPC atômicas)
